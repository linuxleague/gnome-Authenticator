## Features

- Time-based/Counter-based methods support
- SHA1/SHA256/SHA512 algorithms support
- QR code scanner
- Lock the application with a password
- Beautiful UI
- Backup/Restore from/into known applications like FreeOTP+, andOTP, Bitwarden
